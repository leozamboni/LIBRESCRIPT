Install LibreScript 0.0.1 Alpha

GNU/Linux

Install GCC and Make

sudo apt-get update && apt-get install build-essential

in Arch

sudo pacman -S base-devel

in Red Hat

yum install gcc gcc-c++ make

Install LibreScript

sudo git clone https://github.com/leozamboni/LibreScript && cd LibreScript && make && export PATH="$PATH:$PWD"

Windows

Install MinGW

Install LibreScript

git clone https://github.com/leozamboni/LibreScript & cd LibreScript & mingw32-make & set PATH=%PATH%;%cd%

First program in LibreScript

create hello.libre

import stdio.h;


string hello :3 'Hello, world!';

@clang
  printf("%s", hello);
@end

Compile ancap hello.libre
