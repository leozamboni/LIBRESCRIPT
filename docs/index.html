LibreScript is a free and open source compiled scripting language with native Shell and C programming language libraries support.

Install LibreScript 0.0.1 Alpha

GNU/Linux

Install GCC and Make

sudo apt-get install build-essential

Install LibreScript

sudo git clone https://github.com/leozamboni/LibreScript && cd LibreScript && make && export PATH="$PATH:$PWD"

Windows

Install MinGW

Install LibreScript

Hello world in LibreScript

LibreScript doesn't have your own stdio yet but you can use the @clang operator to use C stdio functions:

import stdio.h;


string hello :3 'Hello, world!';

@clang
printf("%s", hello);
@end

Compile:

librescript hello.libre
